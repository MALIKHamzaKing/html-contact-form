# Simple Contact Us Form 📝

A clean and well-structured HTML form designed as part of my web development learning journey. 

### ✨ Features:
- Built with **Semantic HTML5**.
- Includes various input types (Text, Email, Dropdown, Textarea).
- Uses `required` attributes for basic client-side validation.
- Accessible labels for better user experience.

### 🎯 Goal:
My aim was to understand how data is captured on the frontend before moving toward CSS styling and Backend/Cybersecurity aspects.

### 🛠️ Tech Used:
- HTML5
